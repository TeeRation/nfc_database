# nfc_database
Учебный проект базы данных для учета NFC-меток.

## Структура проекта

schema/     - создание таблиц;
seed/       - тестовые данные;
queries/    - примеры запросов;
data/csv/   - исходные данные;
docs/uml/   - UML-диаграммы;

## Особенности

Все идентификаторы (id) представлены UUID в текстовом формате (TEXT), а не AUTOINCREMENT.

База данных разработана для SQLite.
