# nfc_database
База данных информационной системы учета NFC-меток.

## Особенности проекта

- Все идентификаторы (`id`) представлены в формате UUID (`TEXT`).
- Автоинкремент (`AUTOINCREMENT`) не используется.
- Для обеспечения целостности данных используются внешние ключи (`FOREIGN KEY`).
- Все SQL-скрипты разделены по назначению:
  - `schema/` — создание структуры БД;
  - `seed/` — заполнение тестовыми данными;
  - `queries/` — примеры запросов;
  - `docs/` — документация и UML-диаграммы.
